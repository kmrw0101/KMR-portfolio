## 📁 SQL Toolkit (Reusable Templates)

The **toolkit/** folder contains reusable SQL templates I’ve developed and refined over years of data QA work.  
These scripts form the backbone of how I validate data pipelines, reporting layers, and source‑to‑target mappings.

Included patterns:
- **Test setup templates** — scaffolding for creating test scenarios, sample data, and QA prep  
- **Validation templates** — structured data quality checks and pass/fail logic  
- **Comparison utilities** — `INTERSECT`, `EXCEPT`, row‑count checks, and reconciliation patterns  

These templates are designed to be copied, customized, and reused across projects.
